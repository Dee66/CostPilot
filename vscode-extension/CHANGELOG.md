# Change Log

All notable changes to the "costpilot" extension will be documented in this file.

## [0.1.0] - 2024-01-15

### Added
- Initial release
- Real-time cost diagnostics for Terraform files
- Inline cost estimates next to resources
- Quick fix code actions for cost optimizations
- Cost issue tree view organized by severity
- Status bar with cost summary
- Rich webview reports for analysis results
- Terraform plan analysis command
- Dependency map visualization with Mermaid
- SLO compliance checking
- Workspace scanning
- Context menu integration for .tf and .tfplan files
- Configurable settings for all features

### Features
- Zero-IAM architecture (uses CLI backend)
- Deterministic analysis
- Offline-capable
- Auto-scan on save (optional)
- Problem matcher integration
- Welcome message for first-time users
